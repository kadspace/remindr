v2.2.0 — design language — february 2026

A fast reminder and note app where you type naturally and the app does the schedule parsing. Local-first, dark-mode only, built with Kotlin Multiplatform + Compose.

01 Logo & Mark

Stagger Mark

Two horizontal lines with round caps. Top line full width in #DB504A, bottom line at 70% width in #5A5A5A. Represents a task list — done and in progress.

App icon: 108×108dp adaptive icon viewport on #0E0E0E background. 5dp stroke width.

Wordmark: Inter 700, -1.5px letter spacing, #ECECEC. In-app composable uses SemiBold, -0.5sp.
